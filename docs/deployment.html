Key Request Flows
1
Resume Analysis
Browser → POST /analyze (multipart PDF)
→ spaCy extracts skills
→ Gemini classifies + enriches
→ stored in BigQuery
→ returns candidate_id + trait profile
→ cached in localStorage as analysisData
2
Enrichment Pipeline (~20–30 s)
Browser → POST /enrich
Round 1 (parallel): MarketPulse + SalaryIntel
→ DDG search (2 queries each, parallel)
→ Gemini synthesis
Round 2 (parallel): AutoGap + CompanyRadar
→ cached as reskillio_enrich_{id}
3
Pathway / 90-Day Plan (~15 s)
Browser → GET /pathway/{id}
→ DDG searches Coursera + Udemy for each gap skill (parallel, 6 threads)
→ single Gemini call synthesises 3-phase roadmap
→ cached as reskillio_pathway_{id}
4
Opportunity Radar (~30 s)
Browser → POST /radar/search
→ Adzuna API fetches live jobs
→ Gemini normalises up to 25 jobs (8 parallel threads)
→ scored + ranked for candidate fit
→ shown on radar.html
5
JD Gap Analysis (~10 s)
Browser → POST /gap
→ JD text matched against candidate profile in BigQuery
→ returns exact / transferable / missing skill breakdown
→ gap score computed
→ cached as reskillio_persongap_{id}
6
Weekly Check-in (instant local)
companion.html → slider values saved to localStorage immediately → POST /companion/checkin for backend sync → async digest generation triggered in background thread on Cloud Run
