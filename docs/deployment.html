Key Request Flows
1
Resume Analysis
Browser → POST /analyze (PDF)
→ spaCy NLP extracts skills
→ Gemini classifies + enriches traits
→ stored in BigQuery
→ returns candidate_id + trait profile
→ cached as analysisData in localStorage
2
Enrichment (~20–30 s)
Browser → POST /enrich
Round 1: MarketPulse + SalaryIntel in parallel
→ DDG search (2 queries each, parallel)
→ Gemini synthesis
Round 2: AutoGap + CompanyRadar in parallel
→ cached as reskillio_enrich_{id}
3
90-Day Pathway (~15 s)
Browser → GET /pathway/{id}
→ DDG searches Coursera + Udemy per skill (6 threads)
→ single Gemini call → 3-phase roadmap
→ cached as reskillio_pathway_{id}
4
Opportunity Radar (~30 s)
Browser → POST /radar/search
→ Adzuna API fetches live job listings
→ Gemini normalises up to 25 jobs (8 parallel threads)
→ scored + ranked for candidate fit
5
JD Gap Analysis (~10 s)
Browser → POST /gap
→ JD text matched against profile in BigQuery
→ exact / transferable / missing skill breakdown
→ cached as reskillio_persongap_{id}
6
Weekly Check-in (instant)
companion.html → sliders saved to localStorage immediately
→ POST /companion/checkin for backend sync
→ async Gemini digest generated in background thread
→ locked until next Monday
